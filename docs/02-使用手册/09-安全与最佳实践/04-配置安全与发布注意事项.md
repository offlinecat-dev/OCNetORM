# 配置安全与发布注意事项

## 关键原则
- 密钥、证书、签名材料使用环境变量或安全存储
- 调试日志和敏感字段输出在发布前关闭或脱敏
- 发布前固定配置基线，不要临时改参数

## DatabaseConfig 建议
```ts
import { DatabaseConfig } from 'ocorm'

const config = new DatabaseConfig('app.db')
  .setQueryTimeout(1500)
  .setQueryCache(true, 200, 60000)
  .setMaxConcurrentQueries(4)
```

## 发布检查清单
- 是否关闭不必要的调试日志
- 原生 SQL 是否全部参数化
- 事务超时和重试策略是否明确
- 迁移是否已在预发验证
