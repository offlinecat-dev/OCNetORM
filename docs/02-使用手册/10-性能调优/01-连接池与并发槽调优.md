# 连接池与并发槽调优

## 核心目标
- 降低高并发下的等待时间
- 避免超时和连接争用

## 推荐配置
```ts
import { DatabaseConfig } from 'ocorm'

const config = new DatabaseConfig('app.db')
  .setConnectionPool({ enabled: true, minSize: 1, maxSize: 4, acquireTimeoutMs: 5000 })
  .setMaxConcurrentQueries(4)
  .setQueryTimeout(1500)
```

## 调优顺序
1. 先调 `queryTimeout`
2. 再调 `maxConcurrentQueries`
3. 最后调连接池大小
