# 实体模型与元数据

## 何时用
需要明确字段映射、主键策略、软删除规则时。

## 怎么用
在实体层先定义好字段类型、可空性、默认值和唯一性。

```ts
import { ColumnType, defineEntity } from 'ocorm'

defineEntity('User', {
  tableName: 'users',
  columns: [
    { property: 'id', primaryKey: true, autoIncrement: true },
    { property: 'userName', name: 'user_name', type: ColumnType.TEXT, unique: true }
  ],
  softDelete: true
})
```

## 常见误用
只定义字段名，不定义主键或唯一约束策略。
