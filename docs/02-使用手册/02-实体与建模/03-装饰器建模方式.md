# 装饰器建模方式

## 何时用
团队偏好类声明式模型，且希望注解表达实体约束时。

## 怎么用
保持装饰器模型与 `defineEntity` 二选一，避免重复注册。

```ts
import { Entity, Column, PrimaryGeneratedColumn } from 'ocorm'

@Entity('users')
class User {
  @PrimaryGeneratedColumn()
  id: number = 0

  @Column({ name: 'user_name' })
  userName: string = ''
}
```

## 常见误用
同一实体同时使用装饰器和 `defineEntity`。
