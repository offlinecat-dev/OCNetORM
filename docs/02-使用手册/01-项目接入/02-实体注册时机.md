# 实体注册时机

## 何时用
项目启动阶段需要保证实体元数据可用时。

## 怎么用
先 `defineEntity` 注册实体，再执行 `OCORMInit`。

```ts
import { defineEntity, OCORMInit, DatabaseConfig } from 'ocorm'

defineEntity('User', { tableName: 'users', columns: [] })
await OCORMInit(context, {
  config: new DatabaseConfig('app.db'),
  autoCreateTables: true
})
```

## 常见误用
在第一次仓储调用时才动态注册实体。
