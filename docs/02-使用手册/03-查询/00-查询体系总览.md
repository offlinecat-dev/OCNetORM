# 查询体系总览

## 何时用
开始设计应用查询层时。

## 怎么用
通过 `QueryBuilder` 构造查询，再由 `QueryExecutor` 执行。

```ts
import { QueryExecutor, Repository } from 'ocorm'

const repo = new Repository('User')
const qb = repo.createQueryBuilder().paginate(1, 20)
const page = await new QueryExecutor(qb).getPaginated()
```

## 常见误用
直接拼接 SQL 代替 QueryBuilder，失去守卫和统一语义。
