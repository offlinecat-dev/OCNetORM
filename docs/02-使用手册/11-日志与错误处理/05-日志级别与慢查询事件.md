# 日志级别与慢查询事件

## 推荐日志级别
- `DEBUG`：本地调试
- `INFO`：常规运行
- `WARN`：可恢复异常
- `ERROR`：失败与告警

## 配置示例
```ts
import { DatabaseConfig, LogLevel, OCORMInit } from 'ocorm'

await OCORMInit(this.context, {
  config: new DatabaseConfig('app.db').setQueryTimeout(1500),
  enableLogger: true,
  logLevel: LogLevel.INFO
})
```
