# 术语表

## 何时用
阅读使用手册前先统一核心概念，避免把 API 语义理解错。

## 怎么用
优先按公开 API 解释术语，不依赖仓库内部实现路径。

- `EntityData`：实体数据载体。
- `Repository`：仓储入口，负责 CRUD 与事务封装。
- `QueryBuilder`：链式查询构造器。
- `QueryExecutor`：查询执行器，负责实体模式与原生模式执行。
- `TransactionOptions`：事务选项（超时、重试、隔离级别、只读）。
- `SchemaBuilder`：建表入口。
- `MigrationManager`：迁移执行入口。

## 常见误用
把术语当作内部类路径或测试专用能力使用。
