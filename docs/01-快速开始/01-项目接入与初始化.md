# 项目接入与初始化

## 前置条件
- 已有 HarmonyOS 工程，且可在业务代码中拿到 `Context`（例如 `this.context`）。
- 尚未初始化数据库（避免并发重复初始化）。

## 完整代码
```bash
ohpm i @offlinecat/ocorm
```

```ts
import { Context } from '@kit.AbilityKit'
import { DatabaseConfig, LogLevel, OCORMInit } from 'ocorm'

let initialized = false

export async function initOcorm(context: Context): Promise<void> {
  if (initialized) {
    return
  }

  const config = new DatabaseConfig('app.db')
    .setQueryCache(true, 200, 60000)
    .setQueryTimeout(1500)

  await OCORMInit(context, {
    config,
    autoCreateTables: true,
    enableLogger: true,
    logLevel: LogLevel.INFO
  })

  initialized = true
}
```

## 验证步骤
1. 执行依赖安装：`ohpm i @offlinecat/ocorm`。
2. 在应用启动路径调用一次：`await initOcorm(this.context)`。
3. 再次调用应直接返回，不重复初始化。

## 常见失败
- `ConnectionError`：数据库连接创建失败（文件/权限/环境问题）。
- `DatabaseNotInitializedError`：跳过 `OCORMInit(...)` 就开始 `Repository/QueryExecutor`。
- `ExecutionError`：后续查询超时或 SQL 执行失败，可先下调并发、增大 `setQueryTimeout(...)`。
