# 关系加载与分页

## 前置条件
- 已安装：`ohpm i @offlinecat/ocorm`，导入统一 `from 'ocorm'`。
- 已先初始化数据库，并完成 `User`、`Post` 两个实体注册。
- 关系元数据必须先注册，否则 `with(...)` / `withCount(...)` 会失败。

## 完整代码
```ts
import {
  ColumnType,
  EntityData,
  EntityDataInput,
  MetadataStorage,
  QueryExecutor,
  RelationMetadata,
  RelationType,
  Repository,
  defineEntity
} from 'ocorm'

let relationReady = false

function registerUserAndPost(): void {
  if (relationReady) {
    return
  }

  defineEntity('User', {
    tableName: 'users',
    columns: [
      { property: 'id', name: 'id', primaryKey: true, autoIncrement: true },
      { property: 'userName', name: 'user_name', type: ColumnType.TEXT, unique: true }
    ]
  })

  defineEntity('Post', {
    tableName: 'posts',
    columns: [
      { property: 'id', name: 'id', primaryKey: true, autoIncrement: true },
      { property: 'userId', name: 'user_id', type: ColumnType.INTEGER },
      { property: 'title', name: 'title', type: ColumnType.TEXT }
    ]
  })

  const metadata = MetadataStorage.getInstance()
  if (!metadata.hasRelation('User', 'posts')) {
    metadata.registerRelation(
      'User',
      new RelationMetadata(
        RelationType.ONE_TO_MANY,
        'User',
        'Post',
        'posts',
        'userId'
      )
    )
  }

  relationReady = true
}

export async function queryUserWithPosts(): Promise<void> {
  registerUserAndPost()

  const userRepo = new Repository('User')
  const postRepo = new Repository('Post')

  const userInput = EntityDataInput.create()
  userInput.set('userName', 'relation-demo')
  const userSave = await userRepo.save(EntityData.from('User', userInput))
  if (!userSave.success || userSave.insertId <= 0) {
    throw new Error(`创建用户失败: ${userSave.errorMessage}`)
  }

  const postInput = EntityDataInput.create()
  postInput.set('userId', userSave.insertId)
  postInput.set('title', 'hello relation')
  const postSave = await postRepo.save(EntityData.from('Post', postInput))
  if (!postSave.success) {
    throw new Error(`创建文章失败: ${postSave.errorMessage}`)
  }

  const qb = userRepo.createQueryBuilder()
    .with('posts')
    .withCount('posts', 'postCount')
    .paginate(1, 20)

  const page = await new QueryExecutor(qb).getPaginated()
  if (page.total === 0) {
    throw new Error('关系分页查询失败：未命中用户数据')
  }

  const firstUser = page.data[0]
  const posts = firstUser.getRelatedArray('posts')
  const postCount = firstUser.getTransient('postCount')
  if (posts.length < 1 || Number(postCount) < 1) {
    throw new Error('关系加载或 withCount 结果不符合预期')
  }
}
```

## 验证步骤
1. 执行 `await queryUserWithPosts()`。
2. 结果页应至少有 1 条用户，且首条用户的 `posts.length >= 1`。
3. `firstUser.getTransient('postCount')` 应为大于等于 1 的数值。

## 常见失败
- `RelationNotFoundError`：关系未注册或 `with('posts')` 名称拼写错误。
- `EntityNotRegisteredError`：`User/Post` 任一实体未 `defineEntity(...)`。
- `ExecutionError`：关联加载 SQL 执行失败（通常是 `inverseKey` 写错）。
