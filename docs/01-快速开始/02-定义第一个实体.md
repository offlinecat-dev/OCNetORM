# 定义第一个实体

## 前置条件
- 已完成依赖安装：`ohpm i @offlinecat/ocorm`。
- 你准备通过 `defineEntity(...)` 进行“纯库建模”（非装饰器模式）。

## 完整代码
```ts
import { ColumnType, defineEntity } from 'ocorm'

export function registerUserEntity(): void {
  defineEntity('User', {
    tableName: 'users',
    columns: [
      { property: 'id', name: 'id', primaryKey: true, autoIncrement: true },
      { property: 'userName', name: 'user_name', type: ColumnType.TEXT, unique: true },
      { property: 'age', name: 'age', type: ColumnType.INTEGER, nullable: true }
    ],
    softDelete: true,
    indexes: [
      { columns: ['userName'], unique: true }
    ]
  })
}
```

## 验证步骤
1. 在初始化前调用一次：`registerUserEntity()`。
2. 后续 `new Repository('User')` 不应抛出实体未注册异常。
3. 执行建表后，表 `users` 应包含 `id/user_name/age/deleted_at` 字段。

## 常见失败
- `EntityNotRegisteredError`：没有先执行 `defineEntity('User', ...)`。
- `Error: 注册列 ... 失败`：列配置非法（如类型不正确）。
- `Error: 索引列未注册`：`indexes.columns` 使用了未在 `columns` 中定义的属性/列名。
