# 第一次保存与查询

## 前置条件
- 已先完成实体注册（`defineEntity('User', ...)`）和 `OCORMInit(...)`。
- 当前导入统一来自 `from 'ocorm'`。

## 完整代码
```ts
import {
  ConditionOperator,
  EntityData,
  EntityDataInput,
  QueryExecutor,
  Repository
} from 'ocorm'

export async function saveAndQueryUser(): Promise<void> {
  const repo = new Repository('User')

  const input = EntityDataInput.create()
  input.set('userName', 'trinity')
  input.set('age', 28)

  const saveResult = await repo.save(EntityData.from('User', input))
  if (!saveResult.success) {
    throw new Error(`保存失败: ${saveResult.errorMessage}`)
  }

  const qb = repo.createQueryBuilder()
    .where('age', ConditionOperator.GREATER_EQUAL, 18)
    .paginate(1, 20)

  const page = await new QueryExecutor(qb).getPaginated()
  if (page.total === 0) {
    throw new Error('查询失败：未命中 age >= 18 的数据')
  }

  const first = page.data[0]
  const userName = first.getPropertyValue('userName')
  if (userName !== 'trinity') {
    throw new Error(`数据校验失败: userName=${String(userName)}`)
  }
}
```

## 验证步骤
1. 调用 `await saveAndQueryUser()`。
2. 断言 `saveResult.success === true`。
3. 断言 `page.total > 0` 且首条 `userName === 'trinity'`。

## 常见失败
- `ValidationError`：唯一约束或字段验证不通过（如重复 `userName`）。
- `InvalidConditionError`：`where(...)` 使用了不存在的属性/列名。
- `ExecutionError`：数据库未初始化、SQL 执行失败、或查询超时。
