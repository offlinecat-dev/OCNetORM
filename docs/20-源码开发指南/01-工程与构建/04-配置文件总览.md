# 配置文件总览

## 1. 根配置与模块配置边界
- 根 `build-profile.json5` 管产品、签名、SDK 与模块映射
- 模块 `OCORM/build-profile.json5` 管 HAR 自身构建选项（release 混淆）
- 模块声明与包管理分离：`module.json5` 不等于 `oh-package.json5`

```json5
// build-profile.json5（根，节选）
{
  "app": {
    "products": [
      {
        "name": "default",
        "targetSdkVersion": "6.0.1(21)",
        "compatibleSdkVersion": "5.0.5(17)",
        "runtimeOS": "HarmonyOS"
      }
    ],
    "buildModeSet": [{ "name": "debug" }, { "name": "release" }]
  },
  "modules": [
    { "name": "entry", "srcPath": "./entry" },
    { "name": "ocorm", "srcPath": "./OCORM" }
  ]
}
```

```json5
// OCORM/build-profile.json5（节选）
{
  "apiType": "stageMode",
  "buildOptionSet": [
    {
      "name": "release",
      "arkOptions": {
        "obfuscation": {
          "ruleOptions": { "enable": true, "files": ["./obfuscation-rules.txt"] },
          "consumerFiles": ["./consumer-rules.txt"]
        }
      }
    }
  ],
  "targets": [{ "name": "default" }, { "name": "ohosTest" }]
}
```

```json5
// OCORM/src/main/module.json5（节选）
{
  "module": {
    "name": "ocorm",
    "type": "har",
    "deviceTypes": ["default", "tablet", "phone", "2in1"]
  }
}
```

```json5

  "module": {
    "name": "entry",

```

## 3. 构建命令序列（配置变更后）
```powershell
# 每次改完 build-profile/module/oh-package 后执行
ohpm install
tools\hvigor-local.cmd clean
tools\hvigor-local.cmd assembleHar
tools\hvigor-local.cmd assembleHap
tools\hvigor-local.cmd test
```

## 4. 常见错误与修复命令

### 4.1 签名路径失效（换机器常见）
```powershell
# 查看根签名配置是否还指向当前机器可访问路径
Get-Content build-profile.json5

# 重新构建验证
tools\hvigor-local.cmd clean
tools\hvigor-local.cmd assembleHap
```

### 4.2 `module.type` 被误改导致产物类型异常
```powershell
# 快速核对两个模块声明
Get-Content OCORM\src\main\module.json5
Get-Content entry\src\main\module.json5

# 修复后重建
tools\hvigor-local.cmd assembleHar
tools\hvigor-local.cmd assembleHap
```

### 4.3 依赖与版本信息不一致
```powershell
# 检查库包版本与入口
Get-Content OCORM\oh-package.json5

# 重装依赖并重建
ohpm install
tools\hvigor-local.cmd clean
tools\hvigor-local.cmd assembleHar
tools\hvigor-local.cmd assembleHap
```
