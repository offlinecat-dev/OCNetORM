# 09-分页与排序

分页由 `limit/offset` 或 `paginate/forPage` 驱动，排序由 `orderBy` 驱动。

## 正确示例
```ts
const page1 = await new QueryExecutor(
  repo.createQueryBuilder()
    .orderBy('createdAt', 'DESC')
    .paginate(1, 20)
).getPaginated()
```

```ts
const list = await new QueryExecutor(
  repo.createQueryBuilder()
    .orderBy('id', 'ASC')
    .limit(10)
    .offset(20)
).get()
```

## 误用示例（分页参数非法）
```ts
const qb = repo.createQueryBuilder().paginate(-5, 0)
```

语义结果：不会抛错，参数会被修正为 `page=1,size=20`。

## 误用示例（排序方向非法）
```ts
repo.createQueryBuilder().orderBy('id', 'upward' as never)
```

语义结果：抛 `InvalidConditionError('ORDER BY', '非法排序方向: upward')`。
