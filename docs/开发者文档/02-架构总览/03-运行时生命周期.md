# 运行时生命周期（架构+代码走读）

## 1. 启动生命周期
启动阶段由 `OCORMInit` 驱动到 `DatabaseManager.initialize`，并同步运行时配置（缓存、健康检查、并发槽位）。若连接参数变化，会先 `close()` 再重建连接。`RelationLoader.setInClauseLimit` 在初始化后统一设置关系查询分批上限。

```ts
// database/DatabaseManager.ets
async initialize(context: Context, config: DatabaseConfig): Promise<void> {
  const previousConfig = this.config
  this.handleQueryCacheForConfigChange(previousConfig, config)

  if (this.initialized && this.store !== null && this.config !== null) {
    const sameConnection = this.isSameConnectionConfig(this.config, config)
    const samePool = this.isSamePoolConfig(this.config, config)
    if (sameConnection && samePool) {
      this.config = config
      this.applyRuntimeConfig(config)
      return
    }
  }

  if (this.store !== null || this.storePool !== null) {
    await this.close()
  }
  // ... create store / pool / set flags
}
```

```ts
// query/RelationLoader.ets
static setInClauseLimit(limit: number): void {
  RelationLoader.inClauseLimit = RelationLoader.normalizeInClauseLimit(limit)
}

async loadRelationsParallel(entities: Array<EntityData>, relationNames: Array<string>): Promise<Array<EntityData>> {
  return await this.parallelSupport.loadRelationsParallel(entities, relationNames)
}
```

## 2. 请求到 SQL 的路径样例（读路径）

```ts
// 上层调用示例
import { Repository, QueryExecutor, ConditionOperator } from 'ocorm'

const repo = new Repository('Order')
const qb = repo.createQueryBuilder()
  .where('status', ConditionOperator.EQUAL, 'PAID')
  .groupBy('customerId')
  .having('COUNT(*) > ?', [1])
  .selectRaw(['customerId', 'COUNT(*) as orderCount'])

const rows = await new QueryExecutor(qb).getRaw()
```

```text
请求进入 Repository.createQueryBuilder()
  -> QueryBuilder.where/groupBy/having/selectRaw
  -> QueryExecutor.getRaw()
  -> DatabaseManager.withStoreLease(...)
  -> DatabaseManager.acquireQuerySlot()
  -> queryResultSet / querySql
  -> ResultSetRow 映射
  -> DatabaseManager.releaseQuerySlot()
```

## 3. 事务生命周期样例（写路径）

```ts
// repository/Repository.ets
import { TransactionOptions, IsolationLevel } from 'ocorm'

const options = TransactionOptions.fromConfig({
  isolation: IsolationLevel.READ_COMMITTED,
  timeout: 5000,
  retries: 1,
  readOnly: false
})

await repo.transactionWithOptions(async (txRepo) => {
  await txRepo.save(entityA)
  await txRepo.rawExecute('UPDATE account SET balance = balance - ? WHERE id = ?', [100, fromId])
  await txRepo.rawExecute('UPDATE account SET balance = balance + ? WHERE id = ?', [100, toId])
}, options)
```

```ts
// repository/TransactionManager.ets
async transactionWithOptions(action: TransactionAction, options: TransactionOptions): Promise<TransactionResult> {
  const lease = await this.acquireTransactionLeaseOrThrow()
  const releaseLock = await this.acquireTransactionLock(lease.store)
  try {
    return await this.transactionWithOptionsInternal(lease.store, action, options)
  } finally {
    releaseLock()
    lease.release()
  }
}

private async executeTransactionWithOptions(...): Promise<TransactionResult> {
  store.beginTransaction()
  await action(store)
  store.commit()
  // error -> store.rollBack()
}
```

## 4. 关闭与复位生命周期

```ts
// database/DatabaseManager.ets
async close(): Promise<void> {
  this.stopHealthCheck()
  if (this.storePool !== null) {
    await this.storePool.close()
    this.storePool = null
  }
  if (this.store !== null) {
    await this.store.close()
  }
  this.store = null
  this.initialized = false
  this.healthy = false
}
```

```ts
// repository/TransactionManager.ets
static hasAnyActiveTransaction(): boolean {
  return TransactionManager.activeTransactionCount > 0 || TransactionManager.pendingCleanupCount > 0
}
```
