# 10分钟最小闭环

目标：完成“初始化生命周期 + 首次写入 + 首次查询 + 结果校验”。

## 前置条件
- 已完成 [02-安装与导入约定](./02-安装与导入约定.md)
- 能获取应用 `Context`（例如 `this.context`）
- 确认这是数据库初始化的统一入口（避免多页面重复初始化）

## 一段代码跑通完整链路
```ts
import { Context } from '@kit.AbilityKit'
import {
  ColumnType,
  ConditionOperator,
  DatabaseNotInitializedError,
  DatabaseConfig,
  EntityData,
  EntityDataInput,
  EntityNotRegisteredError,
  OCORMInit,
  QueryExecutor,
  Repository,
  defineEntity
} from 'ocorm'

let initialized = false

async function bootstrapOcorm(context: Context): Promise<void> {
  if (initialized) {
    return
  }

  defineEntity('QuickUser', {
    tableName: 'quick_users',
    columns: [
      { property: 'id', name: 'id', primaryKey: true, autoIncrement: true },
      { property: 'userName', name: 'user_name', type: ColumnType.TEXT, unique: true },
      { property: 'age', name: 'age', type: ColumnType.INTEGER, nullable: true }
    ]
  })

  await OCORMInit(context, {
    config: new DatabaseConfig('quick-start.db').setQueryTimeout(1500),
    autoCreateTables: true
  })

  initialized = true
}

export async function runFirstQueryChain(context: Context): Promise<void> {
  try {
    await bootstrapOcorm(context)

    const repo = new Repository('QuickUser')

    const input = EntityDataInput.create()
    input.set('userName', 'neo')
    input.set('age', 18)
    await repo.save(EntityData.from('QuickUser', input))

    const qb = repo.createQueryBuilder()
      .where('userName', ConditionOperator.EQUAL, 'neo')
      .limit(1)

    const row = await new QueryExecutor(qb).getOne()
    if (row === null) {
      throw new Error('首个查询未命中数据')
    }

    const name = row.getPropertyValue('userName')
    if (name !== 'neo') {
      throw new Error('查询结果与写入值不一致')
    }
  } catch (error) {
    if (error instanceof DatabaseNotInitializedError) {
      throw new Error('请先执行 OCORMInit(...)，再执行 Repository/QueryExecutor')
    }
    if (error instanceof EntityNotRegisteredError) {
      throw new Error('请先 defineEntity(...)，再 new Repository(...)')
    }
    throw error
  }
}
```

## 验证步骤
1. 在应用启动后调用 `await runFirstQueryChain(this.context)`。
2. 无异常即表示“首个查询成功链路”通过。
3. 如果失败，按异常信息先排查：初始化顺序 -> 实体注册 -> 条件字段名。

## 初始化生命周期约束
- 先 `defineEntity(...)`，再 `OCORMInit(...)`，最后 `Repository/QueryExecutor`。
- `OCORMInit(...)` 建议在应用启动入口只执行一次，不要分散到多个页面生命周期。
- 关闭连接后若继续使用，必须重新初始化（`DatabaseManager.close()` 后旧连接不可复用）。
